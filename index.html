Preview your forms with Form Preview.

Form Preview is a jQuery plugin. Its task is to simplify previewing input from forms.

You can use this wonderful jQuery plugin for previewing user profiles, article or even a shop item card.

To get started, read the README.md document on Form Preview's repository.

VanillaJS version is coming soon.
